## 设计模式——golang实现

> **生活的理想，就是为了理想的生活。                ——张闻天**

本项目实现了一共23种设计模式和一个农场模拟需求变更时使用的5个综合的设计模式的小项目

本项目包括

* 23种设计模式源代码和测试文件
* 23种设计模式的文档
* [7大设计原则文档](./七大设计原则.md)
